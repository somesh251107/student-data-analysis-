# 📊 Student Sales Data Analysis

## 📌 Project Overview
This project analyzes a sales dataset to understand customer behavior, product performance, and revenue trends. The goal is to extract meaningful insights for business decision-making.

---

## 📂 Dataset
The dataset includes:
- Customer details
- Product category and product name
- Quantity and price
- Payment method
- Country
- Purchase date

---

## 🧹 Data Cleaning
- Handled missing values
- Removed duplicates
- Converted data types
- Created new columns (revenue, month, customer segment)

---

## 📈 Analysis Performed

### 🔹 Revenue Analysis
- Total revenue
- Average order value

### 🔹 Product Analysis
- Top-selling products
- Most popular categories

### 🔹 Customer Analysis
- Unique customers
- Repeat customers
- Customer segmentation

### 🔹 Payment Analysis
- Most common payment methods
- Transaction count by payment type

### 🔹 Time-Based Analysis
- Monthly sales trends
- Sales growth patterns

---

## 📊 Visualizations
- Bar charts
- Line charts
- Pie charts
- Histograms
- Scatter plots

---

## 🧠 Key Insights
- Few products generate majority of revenue
- Repeat customers contribute significantly
- Sales show seasonal trends
- Certain payment methods are more popular

---

## 🚀 Tools Used
- Python
- Pandas
- NumPy
- Matplotlib / Plotly

---

## 🎯 Conclusion
The analysis helps in understanding sales patterns and supports data-driven decision-making for improving business performance.
